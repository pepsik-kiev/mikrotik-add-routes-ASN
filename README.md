# Добавляет маршруты для указанной AS на MikroTik-роутер через SSH/SCP.
# Выполняет полный цикл: получение → агрегация → дедупликация → загрузка.
# АРГУМЕНТЫ:
#   1. USER@IP_РОУТЕРА — адрес и учётная запись роутера (например, admin@192.168.88.1)
#   2. AS — номер автономной системы (AS123 или 123)
#   3. КОММЕНТАРИЙ — текст комментария для маршрутов (будет экранирован)
#   4. ШЛЮЗ — gateway (IPv4-адрес или имя интерфейса, например 10.0.0.1 или pppoe-out1)
# ПРИМЕРЫ:
#   $ mikrotik_as_routes.py add admin@192.168.88.1 AS123 "Провайдер" 10.0.0.1
#   $ mikrotik_as_routes.py add admin@192.168.88.1 123 "Облако" vds_wg
